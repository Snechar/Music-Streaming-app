### Set up the project
1.Clone Repository
```bash
git clone https://github.com/Snechar/Music-Streaming-app
```
2.Start the API
```bash
cd Music_Streaming
dotnet build
dotnet run dev
```

3.Install dependencies and run the app
```bash
cd Music-Streaming-Website/music-streaming
npm install
npm start
```
